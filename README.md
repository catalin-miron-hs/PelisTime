# My-Electron-Redux-Boilerplate
Electron application boilerplate based on React, Redux, webpack, bootstrap.

![](./docs/screenshot.png)

## Run

run webpack dev server

```bash
$ npm run start-dev-server
```

and, in another terminal.

run electron

```bash
$ npm run start-dev
```
